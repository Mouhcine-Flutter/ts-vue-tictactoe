## Getting Started

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mouhcine-flutter/ts-vue-tictactoe.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run app locally : http://localhost:3000
   ```sh
   npm run dev
   ```
